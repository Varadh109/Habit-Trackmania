# Habit Trackmania

**Habit Trackmania** is a simple and clean habit-tracking web application designed to help users build consistency, track progress, and achieve daily goals without unnecessary complexity.

Built using **HTML, CSS, and JavaScript**, this project focuses on minimal design, clarity, and productivity.

---

##  Features

*  Add and manage daily habits
*  Weekly progress tracking system
*  Completion rate visualization
*  Daily statistics dashboard
*  Clean and modern dark UI
*  Lightweight — runs directly in the browser
*  No login or installation required

## How to Use

### Method 1 — Online

1. Open the live demo link.
2. Add your habits.
3. Track progress daily.

### Method 2 — Run Locally

1. Download or clone this repository:

   ```bash
   git clone https://github.com/Varadh109/Habit-Trackmania.git
   ```
2. Open `Habit Trackmania  v14 final.html` in your browser.
3. Start tracking habits.
